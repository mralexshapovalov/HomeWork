Параметры по умолчанию //Default parametrs 
Параметр по умолчнанию назаываются параметрами у когорого значение по умолчанию 

int ADD(int a=0,int b=0,int c=0) b и c являются параметрами по умолчани,по скольку у них есть значения по умолчаию 
Значение нельзя дать лишь первому парамету или какому-нибудь параметру из списка аргументов.Компилятор не поймет какое и кому значение передать т.е благодоря параметру по умолчанию функцию можно не до передать параметру
