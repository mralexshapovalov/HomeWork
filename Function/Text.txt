type -тип возвращаемого заначения функции 
любая функуци по завершении возвращает какое-либо значение результиат своей работы 
Функция может вернуть занчение люого существуещего тип 
Некоторые функции ничего не возвращают по завершении,по скольку резульат их работы и так виден и рузульат ее работы можно проверить выводом на экран 
Тип возращаемого значения это тип функции является void

void -постота
return value // value возвращаемое значение 
ключевое слово retuen перрывает работу функциию и возвращает значение функции а также возвращает управление на место вызова
ключевое слово reyurn может вернуть одно значение 
Оно не может вернуть два ли более значение
